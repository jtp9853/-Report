가상의 애완동물 키우기 웹 애플리케이션

기술 스택 및 구현 방법

발표자: [이노학]

날짜: [2024/06/26]

프로젝트 개요

프로젝트 목적:

  • 사용자가 가상의 애완동물을 키우며 재미와 학습을 경험할 수 있도록 합니다.

주요 기능 요약:

  • 애완동물 선택, 먹이 주기, 놀아주기, 훈련, 레벨 업, 아이템 상점, 타 사용자 간 상호작용

주요 기능

  • 애완동물 선택: 랜덤 시스템을 통한 알 선택
  • 먹이 주기: 다양한 먹이 선택 가능 (경험치 반영)
  • 놀아주기: 산책, 공 던져서 받아오기
  • 훈련: 음성 인식을 통한 명령 수행
  • 레벨 업 시스템: 포만감, 똥 치우기, 친밀도
  • 아이템 상점: 기본/고급 사료, 캣타워 등
  • 타 사용자 간 상호작용: 꾸미기 콘테스트, 선물 보내기, 강아지 어질러티

UI 예시

  • 토스의 고양이 키우기
  • 포켓몬스터 도감 관점
  • 닌텐독스

프론트엔드 기술 스택

  • HTML/CSS: UI 구성 및 스타일링
  • JavaScript (ES6+): 로직 처리 및 상태 관리
  • React: 애완동물 선택, 먹이 주기, 놀아주기 등의 기능 구현

백엔드 기술 스택

  • Node.js: 서버 환경 구축
  • Express.js: RESTful API 구현
  • Socket.IO: 실시간 통신 구현

프로젝트 구조

프론트엔드

  • App.js: 메인 애플리케이션 컴포넌트
  • PetList.js: 애완동물 리스트 컴포넌트
  • FeedButton.js: 먹이 주기 버튼 컴포넌트
  • PlayButton.js: 놀아주기 버튼 컴포넌트

백엔드

  • index.js: 서버 진입점
  • controllers/petController.js: API 요청 처리 컨트롤러
  • models/Pet.js: 데이터베이스 모델
  • routes/petRoutes.js: API 라우터
  • utils/socket.js: 실시간 통신 설정

구현 방법

RESTful API 설계

Express.js를 사용하여 `/pets`, `/feed`, `/play` 등의 엔드포인트를 구현합니다. 클라이언트에서 요청한 애완동물 데이터를 처리하고 상태를 업데이트합니다.

Socket.IO를 이용한 실시간 통신

Node.js 서버에서 Socket.IO를 사용하여 클라이언트와의 양방향 통신을 설정하고, 애완동물 상태 변경 시 실시간 업데이트를 클라이언트에게 전송합니다.

결론

가상의 애완동물 키우기 웹 애플리케이션은 사용자가 애완동물을 키우는 경험을 가상 환경에서 즐길 수 있도록 합니다. 다양한 기능과 실시간 상호작용을 통해 사용자들에게 재미와 학습을 제공합니다.
